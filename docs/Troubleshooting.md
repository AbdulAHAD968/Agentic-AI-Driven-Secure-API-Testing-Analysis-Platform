# Comprehensive Troubleshooting & Error Resolution Guide

## 1. Network & Connectivity Errors
### ERR_CONN_REFUSED
- **Description:** The backend agent cannot reach the target API.
- **Resolution:** Check if the target API is behind a firewall or if the URL is correct.
- **Log Pattern:** `[ERROR] [AGENT_AUTH] Connection refused at 0x00451`

### ERR_SSL_HANDSHAKE_FAILURE
- **Description:** Occurs when testing APIs with self-signed certificates.
- **Resolution:** Enable the `SKIP_SSL_VERIFY` flag in the Agent configuration.

## 2. AI Agent Reasoning Failures
### AGENT_LOOP_DETECTED
- **Description:** The agent is repeating the same attack pattern without new results.
- **Resolution:** Clear the `agent_memory` cache and restart the task.

### AGENT_MODEL_TIMEOUT
- **Description:** The LLM backend took too long to generate a testing strategy.
- **Resolution:** Check API quota for the underlying model (GPT-4/Claude/Llama).

## 3. Database & Storage Errors
### DB_QUERY_LATENCY_HIGH
- **Description:** Result aggregation is taking longer than 200ms.
- **Resolution:** Optimize the `results` table indices in MongoDB/PostgreSQL.
